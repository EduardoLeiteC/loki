AzCpfCnpj

O azCpfCnpj é utilizado para formatar os valores de CPF ou CNPJ.

Utilização

HTML

<p>CPNJ: {{ cnpj | azCpfCnpj }}</p>
<p>CPF: {{ cpf | azCpfCnpj }}</p>

Vue.js

export default {
    data() {
        return {
            cnpj: '90573070000121',
            cpf: '24940622096'
        }
    }
}
